# go-gin-api
Backend in Go using the Gin framework to generate REST API

Особенности реализации/b
GORM ORM - полная поддержка MySQL с автоматическими миграциями
Пул соединений - оптимизированная работа с БД
Soft delete - поддержка мягкого удаления
Пагинация - эффективная загрузка данных
Транзакции - поддержка транзакций для сложных операций
Конфигурация через .env - безопасное хранение sensitive данных
Docker поддержка - контейнеризация для легкого развертывания
Валидация данных - на уровне DTO и БД
Уникальные индексы - гарантия целостности данных
Логирование запросов - для отладки и мониторинга
